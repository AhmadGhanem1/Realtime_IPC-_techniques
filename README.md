# IPC Techniques under linux (POSIX) and C
Overview

This project demonstrates Inter-Process Communication (IPC) techniques using various C programs. The project consists of multiple modules that work together to simulate real-time data exchange, message passing, and synchronization.

Features

Implementation of IPC mechanisms such as message queues.

Real-time simulation using OpenGL.

Organized modular structure with header and source files.

Compilation using a makefile for easy execution.
